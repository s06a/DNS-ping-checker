# Find Fastest DNS

```bash
# bash DNS <seconds or count>

# usage
bash DNS 2 # waits two seconds and also uses 2 as the ping count

# output
# Top 5 DNS servers
# 8.8.4.4          49.758
# 8.8.8.8          66.830
# 9.9.9.10         71.550
# 149.112.112.10   77.223
# 9.9.9.9          86.227
```

also, you can find a list of available DNS servers in `dnsList` file.
